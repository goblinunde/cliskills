# Bilingual Patterns

## Common Layouts

- source paragraph followed by translated paragraph
- side-by-side columns for short text
- mirrored sections with matching headings
- bilingual abstract, captions, or keywords only

## Selection Guidance

- Use sequential blocks for long-form papers.
- Use side-by-side layout for short notes, handouts, or comparisons.
- Use partial bilingual output when only abstract, captions, or slides need pairing.

## Alignment Rules

- Keep section order identical.
- Keep terminology mappings stable.
- Preserve equations and references rather than duplicating unstable labels.

## Hand-off Guidance

- For one-way translation only, use `latex-tex-translate`.
- For translation plus style improvement, follow with `latex-academic-polish`.
