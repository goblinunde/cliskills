# Beamer Translation Rules

## Slide-First Principle

- Preserve speakability, not sentence-by-sentence literal fidelity.
- Shorten titles that become too long in the target language.
- Split frames if translation makes bullets unreadable.

## Preserve

- frame boundaries unless overcrowding forces changes
- commands, overlays, equations, references, and labels
- speaker notes only when the user wants them translated

## Common Adjustments

- paragraph text -> bullets
- long bullets -> shorter bullets plus oral explanation
- overloaded comparison slide -> two frames

## Final Check

- Are titles still scannable?
- Did any overlay become visually awkward?
- Did text expansion break columns or blocks?
